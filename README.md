📦 Smart Warehouse Grid Locator (Java OOP Project)
📌 Description

The Smart Warehouse Grid Locator is a Java-based console application developed using Object-Oriented Programming (OOP) concepts. It simulates a warehouse management system where items are stored in a 2D grid structure, allowing users to efficiently locate, add, and manage inventory.

This project demonstrates how real-world warehouse systems can be modeled using arrays and object-oriented design, making it ideal for beginners learning Java.

✨ Features
🗂️ 2D grid-based storage system
📍 Locate items using row and column positions
➕ Add items to specific grid locations
🔍 Search items by ID or name
📊 Display warehouse layout
🧠 Beginner-friendly OOP implementation
🛠️ Technologies Used
Java
Object-Oriented Programming (OOP)
